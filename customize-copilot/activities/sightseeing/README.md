# City Sightseeing Tour - Barcelona

## About this tour

Barcelona's most iconic landmarks in one tour! Visit the Sagrada Familia, Park Güell, and La Rambla.

## Tour Information

- Where: Barcelona
- How long: 3 hours
- Cost: 30€ per person
- For ages: 5+

## Stops

1. **Sagrada Familia** - Gaudí's masterpiece
2. **Park Güell** - Colorful mosaic park
3. **La Rambla** - Famous street with shops and cafés
4. **Gothic Quarter** - Medieval streets and architecture

## Extras

- Guide speaks English and Spanish
- Comfortable shoes recommended
- Water and snacks not included
