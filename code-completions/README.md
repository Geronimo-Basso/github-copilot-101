# Lab 01 — Code Completions

Learn how GitHub Copilot provides **inline code suggestions** as you type. This is the core Copilot experience — writing a comment or function signature and letting Copilot complete the rest.

## What You'll Learn

- How to trigger and accept inline suggestions.
- Writing effective comments to guide completions.
- Navigating between alternative suggestions.
