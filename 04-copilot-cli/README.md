# Lab 04 — Copilot CLI

Get AI-powered help directly in your **terminal**. Copilot CLI can explain commands, suggest shell one-liners, and assist with Git operations.

## What You'll Learn

- Using `gh copilot suggest` and `gh copilot explain`.
- Generating shell commands from natural language descriptions.
- Working with Git commands through Copilot.
