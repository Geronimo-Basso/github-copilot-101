# GitHub Copilot 101

A hands-on repository to learn how to use GitHub Copilot step by step.

## What is this?

This repo contains practical workshops and exercises designed to help you get started with GitHub Copilot — from basic code completions to advanced features like Copilot Chat, custom instructions, and extensions.

## Who is this for?

- Developers new to GitHub Copilot
- Teams looking for structured onboarding material
- Anyone who wants to explore AI-assisted development

## Getting Started

Browse the folders in this repository. Each one contains a self-contained workshop with its own README and instructions. Pick the one that interests you and follow along.

## Prerequisites

- A GitHub account with access to GitHub Copilot
- A supported IDE (VS Code recommended)
- Basic programming knowledge
