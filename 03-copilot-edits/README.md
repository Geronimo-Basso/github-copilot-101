# Lab 03 — Copilot Edits

Use **Copilot Edits** to make changes across multiple files using natural language instructions.

## What You'll Learn

- Starting an edit session and adding files to the working set.
- Describing changes in plain English and reviewing diffs.
- Iterating on edits with follow-up instructions.
