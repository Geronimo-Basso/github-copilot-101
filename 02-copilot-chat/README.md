# Lab 02 — Copilot Chat

Explore **GitHub Copilot Chat** — your AI pair programmer that lives right inside the IDE. Ask it to explain code, fix bugs, write tests, and more.

## What You'll Learn

- Using slash commands (`/explain`, `/fix`, `/tests`).
- Providing context with `#file`, `#selection`, and `@workspace`.
- Generating code through conversational prompts.
